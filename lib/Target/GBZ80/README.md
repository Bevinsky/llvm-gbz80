# GBZ80 backend

This backend is for the 8-bit GBZ80 (Sharp LR35902) CPU, found in the Nintendo Game Boy.

## Use instructions

To compile a C file, invoke clang like this:
```
clang -target gbz80 -S -O3 <c-file>
```
